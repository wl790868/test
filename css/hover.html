测试hovering 雷柯特aaa

小结：

css hover生效条件

  1. chrome ,ie7及以上浏览器：不管a标签还是其它标签均可生效
  2. ie6：只有带href属性的a标签可以
  3. ie6：想定义带href属性的a标签hover时，字节点的样式，必须先定义 a标签:hover 样式。比如：
    /*直接这样是不行的*/ .selected-item:hover .btn-X{ color: green; }
    /*要这样*/ .selected-item:hover{ color:red; } .selected-item:hover .btn-X{ color: green; }
